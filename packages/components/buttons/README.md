# @xaui/buttons

Button components for XAUI - A modern React Native UI library with Flutter-inspired API, smooth animations, and comprehensive design system.

## Installation

```bash
pnpm add @xaui/buttons
```

## Features

- **Flutter-inspired API**: Button components with intuitive Flutter-like props
- **Theme Integration**: Seamlessly integrates with @xaui/core theme system
- **Smooth Animations**: Built on React Native Reanimated for native performance
- **TypeScript First**: Fully typed for excellent developer experience
- **Customizable**: Extensive customization options while maintaining simplicity
